Objectives

Your task is to create the authentication for the service. The service needs two pages to authenticate users: signup and login. On the signup page, the users register their accounts, and on the login page they pass the authentication by their username and password.

Please use django.contrib.auth.forms.UserCreationForm for your signup handler and django.contrib.auth.forms.AuthenticationForm for the login handler.

If the login request is successful, redirect the user to the menu page.

If you start the application on your computer with python manage.py runserver command, the signup page should be available at the address localhost:8000/signup and login on localhost:8000/login.

Pay attention, that the login and signup URLs do not contain a slash at the end, so your URL patterns should not too. You can redirect GET requests to the right page with help or RedirectView if you need to:

from django.views.generic import RedirectView
 
urlpatterns = [
   # some other patterns
   path('login/', RedirectView.as_view(url='/login'))
   path('signup/', RedirectView.as_view(url='/signup'))
]
